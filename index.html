Alerta Ciudadana
Sistema de Emergencia y Rastreo Ciudadano

Aplicación Oficial

Alerta Ciudadana es una aplicación desarrollada para permitir a los ciudadanos enviar alertas de emergencia en tiempo real al equipo de serenazgo.

El sistema comparte ubicación GPS en tiempo real para facilitar la rápida atención y seguimiento de incidentes reportados.

📥 DESCARGAR APK

📱 ¿Qué hace la aplicación?

La aplicación ahora se encuentra en fases de prueba cualquier bug detectado o errores se agradece reportarlos

Cuando ocurre una emergencia, la ubicación del ciudadano es enviada en tiempo real al equipo de seguimiento el cual llamara a la persona para corroborar que sea una alerta real.

🔐 Permisos Necesarios

🗺️ Acceso a ubicación GPS
🌐 Conexión a internet o datos móviles
📞 Información básica del usuario
🔔 Notificaciones y alertas
⚠️ La aplicación requiere mantener activada la ubicación y conexión a internet para garantizar el correcto funcionamiento del seguimiento en tiempo real.

🚨 Beneficios

⚡ Atención rápida
📍 Rastreo en tiempo real
🏛️ Comunicación directa con la municipalidad
🔒 Sistema seguro y monitoreado
